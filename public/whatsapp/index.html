WhatsApp Specifications

Specs and technical requirements for all WhatsApp content formats.

Status Image: 1080 × 1920 px recommended·Status Video: Up to 60 seconds·Message: 65,536 characters

Image Specs

Status Image

Dimensions1080 × 1920 px recommended
Aspect Ratio9:16
Max File Size16 MB
File FormatsJPG, PNG
NotesFull-screen vertical format; status disappears after 24 hours

Profile Photo

Dimensions500 × 500 px recommended
Max File Size5 MB
File FormatsJPG, PNG
NotesDisplayed as a circle; minimum 192 × 192 px

Shared Photo (Chat)

Max File Size16 MB
File FormatsJPG, PNG, GIF, WEBP
NotesWhatsApp compresses images on send; send as Document to preserve quality

Business Catalog Image

Dimensions640 × 640 px recommended
Max File Size5 MB
File FormatsJPG, PNG
NotesWhatsApp Business product/service catalog listings
[Ad]

Video Specs

Status Video

Max File Size16 MB
File FormatsMP4, MOV
Max DurationUp to 60 seconds
NotesDisappears after 24 hours; H.264 codec recommended

Shared Video (Chat)

Max File Size16 MB (regular) or 2 GB (as Document)
File FormatsMP4, MOV, AVI
Max DurationUp to 3 minutes (regular) or 2 GB (as Document)
NotesSending as Document bypasses compression and size limits

View Once Media

Max File Size16 MB
File FormatsJPG, PNG, MP4
NotesCan only be viewed once by the recipient
